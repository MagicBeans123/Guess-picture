# 你画我猜，面向中小学生科普网页
---
## 简介
---

> 以你画我猜的形式切入中小学生AI科普知识，寓教于乐。调用智谱大模型api，自创“蜗壳小助手”形象。

## 本地运行
---

- 申请智谱apikey，申请地址(https://open.bigmodel.cn/usercenter/apikeys)
- 删除.env.example的.example后缀，在.env的apikey位置写入申请的apikey
- 可以自行修改端口号
- 下载python 3.13.7版本
- 双击运行setup.bat，之后点击run.bat
- 访问localhost:5000(或者自设端口号)
